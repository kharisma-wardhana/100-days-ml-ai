# Days 11 - 20 : Pengantar Machine Learning

Ini adalah rencana pembelajaran untuk hari 11 hingga hari 20 dalam rangkaian 100 hari belajar Machine Learning dan Kecerdasan Buatan (AI). Fokus pada tahap ini adalah memahami konsep dasar Machine Learning serta teknik-teknik yang umum digunakan.

## Apa itu Machine Learning?

Machine Learning (ML) adalah cabang dari kecerdasan buatan (AI) yang memungkinkan komputer untuk belajar dari data dan membuat prediksi atau keputusan tanpa diprogram secara eksplisit untuk tugas tersebut.

## Tipe-tipe Machine Learning

1. **Supervised Learning**: Model dilatih menggunakan data berlabel (input-output pairs). Contoh: klasifikasi, regresi.
2. **Unsupervised Learning**: Model dilatih menggunakan data tanpa label. Contoh: clustering, asosiasi.
3. **Reinforcement Learning**: Model belajar melalui interaksi dengan lingkungan untuk memaksimalkan reward.

## Proses Machine Learning

1. **Pengumpulan Data**: Mengumpulkan data yang relevan untuk masalah yang ingin diselesaikan.
2. **Preprocessing Data**: Membersihkan dan menyiapkan data untuk analisis.
3. **Pemilihan Model**: Memilih algoritma machine learning yang sesuai.
4. **Pelatihan Model**: Melatih model menggunakan data latih.
5. **Evaluasi Model**: Mengukur kinerja model menggunakan data uji.
6. **Penyempurnaan Model**: Mengoptimalkan model untuk meningkatkan kinerja.
7. **Deployment**: Menerapkan model ke lingkungan produksi untuk digunakan dalam aplikasi nyata.
8. **Monitoring dan Pemeliharaan**: Memantau kinerja model dan melakukan pembaruan jika diperlukan.
9. **Iterasi**: Proses ini bersifat iteratif, di mana model dapat terus disempurnakan berdasarkan hasil evaluasi dan feedback dari pengguna.
10. **Dokumentasi**: Mencatat proses, keputusan, dan hasil untuk referensi di masa depan.
